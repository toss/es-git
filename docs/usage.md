# Installation & Usage

es-git is available via [npm](https://npmjs.com/package/es-git).

## Node.js

es-git supports Node.js 10.7 or later. Install es-git with the following command:

::: code-group

```sh [npm]
npm install es-git
```

```sh [pnpm]
pnpm add es-git
```

```sh [yarn]
yarn add es-git
```

:::
