# iter

트리 내 항목들을 순회할 수 있는 반복자(iterator)를 반환해요.

## 시그니처

```ts
class Tree {
  iter(): TreeIter;
}
```

### 반환 값

<ul class="param-ul">
  <li class="param-li param-li-root">
    <span class="param-type">TreeIter</span>
    <br>
    <p class="param-description">이 트리의 항목들을 순회하는 반복자(iterator) 객체예요.</p>
  </li>
</ul>
