# isEmpty

트리에 항목이 비어있는지 확인해요.

## 시그니처

```ts
class Tree {
  isEmpty(): boolean;
}
```

### 반환 값

<ul class="param-ul">
  <li class="param-li param-li-root">
    <span class="param-type">boolean</span>
    <br>
    <p class="param-description">트리에 항목이 하나도 없으면 <code>true</code>를 반환하고, 하나라도 있으면 <code>false</code>를 반환해요.</p>
  </li>
</ul>
