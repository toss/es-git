# 클래스: DiffStats

Diff의 하나의 변경 묶음을 나타내는 클래스예요.

## 접근자

### filesChanged

#### get 시그니쳐

> **get** **filesChanged**(): `bigint`

Diff에서 변경된 전체 파일 수를 가져와요.

##### 반환 형식:

`bigint`

***

### insertions

#### get 시그니쳐

> **get** **insertions**(): `bigint`

Diff에서 추가된 전체 라인 수를 가져와요.

##### 반환 형식:

`bigint`

***

### deletions

#### get 시그니쳐

> **get** **deletions**(): `bigint`

Diff에서 삭제된 전체 라인 수를 가져와요.

##### 반환 형식:

`bigint`
