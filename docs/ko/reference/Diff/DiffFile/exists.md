# exists

해당 쪽의 파일이 존재하면 `true`를 반환해요.

## 시그니처

```ts
class DiffFile {
  exists(): boolean;
}
```

### 반환 값

<ul class="param-ul">
  <li class="param-li param-li-root">
    <span class="param-type">boolean</span>
    <br>
    <p class="param-description">파일이 존재하면 <code>true</code>를 반환해요.</p>
  </li>
</ul>
