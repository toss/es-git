# 열거형: RevparseMode

`revparse`에서 사용하는 플래그들이에요.

## 포함된 값

| 포함된 값                              | 값   | 설명                                    |
|------------------------------------|-----|---------------------------------------|
| <a id="single"></a> `Single`       | `1` | 하나의 개체를 가리키는 경우예요.                    |
| <a id="range"></a> `Range`         | `2` | 여러 커밋을 포함하는 범위를 가리키는 경우예요.            |
| <a id="mergebase"></a> `MergeBase` | `4` | `...` 연산자를 사용해 병합 기반(공통 조상)을 찾는 경우예요. |
