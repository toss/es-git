# 열거형: DiffFlags

## 포함된 값

| 포함된 값                              | 값   | 설명                   |
|------------------------------------|-----|----------------------|
| <a id="binary"></a> `Binary`       | `1` | 파일이 바이너리 데이터로 취급     |
| <a id="notbinary"></a> `NotBinary` | `2` | 파일이 텍스트 데이터로 취급      |
| <a id="validid"></a> `ValidId`     | `4` | `id` 값이 유효           |
| <a id="exists"></a> `Exists`       | `8` | 파일이 이 쪽 면의 delta에 존재 |
