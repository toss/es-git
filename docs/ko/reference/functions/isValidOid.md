# 함수: isValidOid()

> **isValidOid**(`value`): `boolean`

Check if given string is valid Oid.

Returns `false` if the string is empty, is longer than 40 hex
characters, or contains any non-hex characters.

## 매개변수

| 매개변수 | 유형 |
| ------ | ------ |
| `value` | `string` |

## 반환 형식:

`boolean`
