# isShallow

이 리포지토리가 얕은 클론(shallow clone)인지 확인해요.

## 시그니처

```ts
class Repository {
  isShallow(): boolean;
}
```

### 반환 값

<ul class="param-ul">
  <li class="param-li param-li-root">
    <span class="param-type">boolean</span>
    <br>
    <p class="param-description">리포지토리가 얕은 클론이면 <code>true</code>를 반환해요.</p>
  </li>
</ul>
