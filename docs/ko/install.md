# 설치하기

es-git은 [npm](https://npmjs.com/package/es-git)에서 설치할 수 있어요.

## Node.js

es-git은 Node.js 10.7 또는 이후 버전을 지원해요. es-git을 설치하기 위해서는 아래 명령어를 사용해주세요.

::: code-group

```sh [npm]
npm install es-git
```

```sh [pnpm]
pnpm add es-git
```

```sh [yarn]
yarn add es-git
```

:::
